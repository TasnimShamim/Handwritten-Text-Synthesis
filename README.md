# Handwritten Text Synthesis ✍️

This project generates human-like handwritten text using deep learning models.

## Features
- Sequence modeling using RNN / LSTM
- Character-level text generation
- Realistic handwriting synthesis

## Tech Stack
- Python
- TensorFlow / PyTorch
- NumPy
- OpenCV

## How to Run
1. Install dependencies
2. Train the model
3. Generate handwriting samples
